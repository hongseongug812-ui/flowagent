핵심 기능

필요한 건 다 있어요

🤖
AI Agent 노드
GPT-4o를 워크플로우 안에 바로 연결. 프롬프트만 입력하면 텍스트 분석, 요약, 분류, 생성까지 자동으로.
🔗
API Call 노드
Slack, Notion, Google Sheets, 카카오, 어떤 API든 코드 없이 연결. GET/POST/PUT 모두 지원.
🔀
Condition 노드
AI 분석 결과에 따라 다른 경로로 분기. "카테고리가 환불이면 A팀, 기술 문의면 B팀".
⚡
Webhook 트리거
URL 하나로 어디서든 워크플로우 실행. Shopify, Gmail, Typeform — 연결 안 되는 곳이 없음.
⏰
스케줄 실행
매일 아침 9시, 30분마다, 매주 월요일. Cron으로 원하는 시간에 자동 실행.
🔄
Transform 노드
데이터 포맷 변환, 필터링, 계산. JavaScript 코드 몇 줄로 원하는 형태로 가공.
